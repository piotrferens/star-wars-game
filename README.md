## Star wars game

### Play the game:

Open [https://star-wars-game.vercel.app/](https://star-wars-game.vercel.app/) with your browser.

### To run game locally

Install all dependencies:

```bash
npm i
```
Run app with:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
